## Tech Stack

Python, FastAPI, LangChain, LangGraph, Qdrant, pgvector, Redis, sentence-transformers, cross-encoder, OpenTelemetry, Docker, Kubernetes

## Prerequisites

- Python 3.10+
- Docker
- kubectl + kind (for K8s labs)
- API key for Claude or OpenAI
